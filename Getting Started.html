RBXe-Framework


Getting Started

RBXe Game Framework is a way of including all your popularly well used functions, and other extras to make developing easier in one little package.
While it is not currently available, this page is reserved as a placeholder.

Basic usage: 


Level = require(SampleID)

You can also grab the model here.
This is the method you should use if you need a LocalScript to use my framework. 

Level = require(Instance ModuleScript)

This is due to ROBLOX not allowing clients to require ModuleScripts by ID, for good reason too.
I do however recommend that you use the framework entirely server side.

Good luck, and enjoy your shorter code!
