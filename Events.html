RBXe-Framework


Events

The Event framework is extremely useful when working with multiple scripts to prevent loops, and an over-flow of instances, increasing load times.

Create

The Create method takes one argument, and returns the event. Basic usage: 

MyEvent = Framework.Events:Create(String Name)

Fire

The Fire method has one required argument, and can have as many extra arguments as you feel like. Basic usage: 

Framework.Events:Fire(String Name (Name used in :Create()), [Args, ...] )

Connection

For your events to actually do something, you'll need to connect them. The syntax should make you feel comfortable. It can be connected in two ways. Basic usage: 

MyEvent:connect(function)

Or it can be connected like so: 
MyEvent:connect(function(Args)
-- Do stuff
end)

Combine

So, now that you know how it all works, it's time to put it all together. 

MyEvent = Framework.Events:Create("Test")

MyEvent:connect(function(text)
  print(text)
end)

Framework.Events:Fire("Test", "Hello World!")
