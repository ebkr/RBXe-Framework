OnTeam, Spawned, ToolEquipped, Respawn

Players

The Players framework contains a few functions that could prove helpful.

OnTeam

OnTeam returns a list of all players on a specific team.
Basic usage: 

TeamRed = Framework.Players:OnTeam(String Name or Instance Team)

Spawned

Spawned returns a list of all players with a currently spawned character.
Basic usage: 

SpawnedPlayers = Framework.Players:Spawned()

ToolEquipped

Returns the current tool equipped by the specified Player or Character.
Basic usage: 

HoldingTool = Framework.Players:ToolEquipped(Instance Player or Instance Model)

Respawn

Can respawn all players in a table. If the table parameter isn't added, it defaults to all players.
Basic usage: 

Framework.Players:Respawn(Optional Player Table)
