RBXe-Framework


Light

The Light framework isn't useful for the majority of games*
It is intended for use with games which depend on measuring volumes of light.

It does however, only work with PointLights, and can't detect obstructions.

Basic usage: 

Level = Framework.Light:DetectLevel(Vector3 Position[Optional, Instance Parent __ Defaults to Workspace])

It is recommended to use DetectLight in Regions. Using recursion through the entire workspace can cause excessive lag in larger games.
